# 歲入款項暨單據管理系統

一個基於 Vue 3 + Vite + TailwindCSS 的前端原型系統，用於展示「歲入款項暨單據管理系統」的功能。

## 公司資訊

**米樂睞資訊 MillerLight Info**

系統設計靈感來自 Miller Lite 啤酒品牌，追求清爽、明亮、簡約的使用者介面體驗。

## 技術堆疊

- **前端框架**: Vue 3 (Composition API + `<script setup>`)
- **建置工具**: Vite
- **路由管理**: Vue Router 4
- **樣式系統**: TailwindCSS
- **狀態管理**: Pinia
- **語言**: JavaScript (非 TypeScript)

## 功能特色

### 🏠 儀表板
- 系統概覽統計
- 單據狀態分佈
- 月度趨勢圖表
- 快速操作入口

### 📝 單據管理
- **單據開立**: 建立新收入單據
- **單據查詢**: 搜尋和管理所有單據
- 支援狀態篩選 (未核銷/已核銷/作廢)

### 💰 核銷作業
- **核銷匯入**: 模擬金融機構繳費明細匯入
- **異常處理**: 處理金額不符與單據遺失案例
- **退費作業**: 處理已核銷單據的退費申請

### ⚙️ 系統管理
- **批次作業**: 監控系統批次處理狀態
- **使用者管理**: 管理帳號、角色與權限

## 快速開始

### 安裝依賴
```bash
npm install
```

### 啟動開發伺服器
```bash
npm run dev
```

專案將在 `http://localhost:5173` 啟動

### 建置生產版本
```bash
npm run build
```

## 測試帳號

**帳號**: `demo`  
**密碼**: `demo`  
**角色**: 可選擇承辦人、稽核人員或業者

## 主要功能流程

### 1. 登入系統
使用測試帳號登入，選擇適當的角色

### 2. 查看儀表板
瀏覽系統整體狀況與統計數據

### 3. 建立單據
在「單據開立」頁面新增收入單據

### 4. 核銷匯入
在「核銷匯入」頁面模擬銀行明細匯入與比對

### 5. 處理異常
在「異常清單」頁面處理匯入後的異常案例

### 6. 執行退費
在「退費作業」頁面處理已核銷單據的退費申請

## 技術亮點

- **響應式設計**: 使用 TailwindCSS 實現清爽的介面
- **模組化組件**: 可重用的 Vue 組件架構
- **狀態管理**: Pinia 統一管理應用狀態
- **假資料模擬**: 完整的 Mock API 與資料流
- **路由守衛**: 基於登入狀態的頁面保護

## 注意事項

- 這是前端原型系統，所有資料皆為假資料
- 無真實的後端 API 串接
- 登入驗證僅為展示用途
- 檔案上傳功能僅為 UI 展示
